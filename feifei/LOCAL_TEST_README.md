本地测试说明

- 目标：在本地通过 127.0.0.1 访问项目页面以做快速测试。

快速运行（推荐）：
1. 在文件资源管理器或终端中进入项目根目录（包含 `index.html` 的目录）。
2. 运行批处理（Windows 命令提示符）：

```bat
serve.bat
```

或在 PowerShell 中运行：

```powershell
./serve.ps1
```

或者直接使用 Python 3（若已安装）：

```powershell
python -m http.server 8000
```

访问测试链接：

- http://127.0.0.1:8000/

说明：服务器会把当前目录作为根目录来提供静态文件服务，打开上面的链接后即可访问 `index.html`。如需更改端口，把 `8000` 替换为你要的端口。
