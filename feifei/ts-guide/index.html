TeamSpeak 使用教程

如何连接服务器并开始语音

什么是 TeamSpeak

TeamSpeak 是一款专注于低延迟语音通话的软件，广泛用于游戏公会、社区语音服务器和在线协作。 相较于其他语音工具，TS 对网络要求更低、延迟更小、隐私保护更好，服务器由管理员自行托管，数据更安全。

超低延迟
数据私有
多平台支持
高度可定制
下载 TeamSpeak

选择你的平台，前往官方页面下载并安装 TeamSpeak 客户端。

Windows ts3 64 位安装包
macOS ts3 Mac 客户端
TS3 汉化下载与安装（可选）

如果你看英文界面不习惯，可以下载安装中文语言包。安装后重启 TS3 即可生效。

下载 TS3 简体中文语言包 Chinese Simplified (zh-CN).ts3_translation
1

先关闭 TeamSpeak 3

安装语言包前先退出 TS3，避免安装后不生效。

2

下载语言包文件

点击上方下载按钮，保存 .ts3_translation 文件到电脑。

3

双击文件安装汉化

下载完成后双击该文件，系统会自动调用 TS3 导入语言包。

4

切换到中文

打开 TS3，进入 Settings → Options → Application，将语言改成 Chinese Simplified。

重启 TS3 检查是否生效

重启后界面应显示中文。如果没生效，再重新双击语言包安装一次。

连接服务器教程

按照以下步骤，即可快速连接到服务器开始语音。

1

打开 TeamSpeak

安装完成后双击打开 TS。第一次打开会让你填 Nickname（昵称），直接填你想让大家看到的名字。

2

点击 Connections

在顶部菜单栏找到 Connections（连接） 菜单并点击展开，或使用快捷键 Ctrl + S。

3

在菜单里点 Connect

在下拉菜单里点 Connect，会弹出一个连接窗口。

4

输入服务器地址

在 Server Address 里填：alecapu.top（注意不要多空格）。

5

输入昵称

在 Nickname 里填你要显示的名字。中文英文都可以，用你常用的 ID 就行。

点击 Connect

点窗口下面的 Connect，等 1-3 秒就会进入服务器。进入后双击频道名字就能进语音房间。

服务器信息

如果你怕打错，可以直接点复制按钮，再粘贴到 TS 的地址输入框。

谱谱大家庭：创建频道教程

本服已开放 游客创建永久频道 权限。按下面步骤操作，就能创建自己的长期频道。

1

先进入服务器

先按前面的连接教程进入 谱谱大家庭 服务器，确保你已经在频道列表里。

2

右键频道列表最上面的谱谱大家庭

在左侧频道列表里，找到谱谱大家庭右键，点击 Create Channel（创建频道）。

3

填写频道名称

在 Channel Name 里填写你想要的频道名，比如：XX的小队语音房。

4

把频道类型设为 Permanent

在频道类型里选择 Permanent（永久频道）。不要选 Temporary（临时），临时频道没人后会自动消失。

5

按需设置密码（可选）

如果你只想让朋友进来，可以勾选频道密码并设置一个简单密码，再发给朋友。

点击 OK 创建完成

点 OK 后频道会立刻出现。以后你再上线，这个永久频道还在。

小提示：请按顺序创建频道，并放到对应的私人频道分类里。请勿乱放频道，如有乱放将先警告一次，第二次将进行删除处理。

连不上？先看这里
  • 先确认地址是 alecapu.top，别多打空格
  • 先关掉 VPN / 代理软件，再试一次
  • 确认电脑网络正常（网页能打开）
  • 如果是校园网/公司网，可能被限制端口，换个网络试试
  • 如仍无法连接，请联系服务器管理员
  • 在 TS 顶部菜单栏点击 Settings → Options
  • 进入 Playback（播放）选项卡，检查输出设备是否为你的耳机/音箱
  • 进入 Capture（采集）选项卡，检查输入设备是否为你的麦克风
  • 确认麦克风的触发方式：Voice Activation（音量检测） 或 Push-to-Talk（按键说话）
  • 检查系统混音器中 TeamSpeak 的音量是否被静音
  • 在频道列表中找到自己的名字，双击即可直接编辑
  • 或在菜单 Settings → Identities 中修改默认昵称

按上面步骤基本都能连上，有问题再找管理员
